# Netty-IM-Demo

---
**中文** | [English](README.en-US.md)


## 简介

`Netty-IM-Demo` 是一个即时通讯的Demo程序，可以作为初学者自学Netty的小实验。

## 功能

- 用户登录和注销
- 用户之间的对话
- 群聊功能（创建/加入/退出群组、列出群聊人数、群聊）

## 你能学到什么

- 如何启动Netty服务端/客户端
- 数据载体 ByteBuf
- 长连自定义协议如何设计
- 粘包拆包原理与实践
- 如何实现自定义编解码
- pipeline 与 channelHandler
- 定时发心跳怎么做
- 如何进行连接空闲检测
